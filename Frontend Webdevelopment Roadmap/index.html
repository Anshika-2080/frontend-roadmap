Frontend Webdevelopment

Frontend Developmetn is the development of the graphical user interface of a wevsite, through the use of HTML, CSS and javaScript, so that users can view and interact with that website.

  • HTML:The Hyper Text Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

  • CSS:Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML

  1. Pure CSS
  2. Bootstrap (Framework)
  3. Tailwind (Framework)
  4. Sass (Processor)

  • JavaScript:JavaScript often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is use to add functionality in the website.

  1. Vanilla JS jQuery (Library)
  2. React (Framework)
  3. Angular (Framework)
  4. Vue(Framework)
  5. Typescript (Preprocessor)

  • Projects:Projects play a important role in the journey of a web developer's learning. It is not just the way to showcase ones skill but also a plateform to work on ones skill.
